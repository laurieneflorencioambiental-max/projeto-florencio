# **App Name**: Florencio Sales Manager

## Core Features:

- Kanban Board: Visualize and manage sales leads through different stages: New, Pending, Approved, Loss, Rejected.
- Drag-and-Drop: Move leads between stages in the Kanban board using drag-and-drop functionality.
- Follow-up Tool: For rejected leads, use AI to analyze the rejection reason and generate a personalized follow-up email or WhatsApp message with discount suggestions.
- Menu: Display a side menu with the company logo, the name 'Comercial Grupo Florencio' and a generic 'gestão' (management) icon.

## Style Guidelines:

- Primary color: Soft teal (#3FB3B3) to give a modern and trustworthy feeling.
- Background color: Light greenish-blue (#E3F4F6), complementing the primary with a softer tone to ensure legibility.
- Accent color: Dark grayish-blue (#596371) for action buttons and highlights to draw attention.
- Headline font: 'Montserrat' (sans-serif) to match the request. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) to match the request. Note: currently only Google Fonts are supported.
- Use a simple set of icons related to sales and management.
- Design a clean layout for the Kanban board to enable an overview of each column.
